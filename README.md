# Production Portfolio Website

This repository contains a complete, production-ready static portfolio website.

## Structure:
- `index.html`: Contains all HTML structure, embedded CSS (implementing responsive design, CSS variables, glassmorphism, and dark mode), and embedded JavaScript for interactivity (theme toggle, preloader).
- `vercel.json`: Configuration file for Vercel deployment.
- `README.md`: This file.

## Features Implemented:
1. Responsive design using CSS Grid/Flexbox (Mobile-first).
2. Professional theming via CSS variables supporting dark/light mode toggle.
3. Modern Glassmorphism effects using `backdrop-filter`.
4. CSS-only loading animation (Preloader).
5. Uses Google Fonts ('Inter').

## One-Click Vercel Deployment:
This project is configured for Vercel. Simply connect this repository to Vercel, and deployment will occur automatically upon pushing changes.
